# pto-ppr-frontend

Фронтенд для проекта PPR на React + Vite + Tailwind.

## Установка

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Переменные окружения

Создай файл `.env`:

```
VITE_API_BASE=https://your-backend-url.onrender.com/api
```

## Деплой

Рекомендован Netlify. Не забудь файл `_redirects`:

```
/*    /index.html   200
```
